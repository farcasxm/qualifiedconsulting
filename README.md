## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Customization

You can customize the website by:
- Updating contact information in `Contact.jsx` and `Footer.jsx`
- Modifying colors in `src/index.css` CSS variables
- Adding/removing services in `Services.jsx`
- Updating content in any component file
